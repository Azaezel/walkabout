The Basics

This tutorial will take you through the very basics of using Walkabout. You'll learn how to add a NavMesh to a mission, build it, and test it with some AIPlayers.

  1. Creating a new mission
  2. Creating a NavMesh
  3. Building the NavMesh
  4. Testing the NavMesh

Creating a new mission

Hit F11 from main menu

Open convex editor

Alt-click to create some boxes

Creating a NavMesh

Open nav editor

Check 'fit to mission' box and hit ok

Building the NavMesh

Make sure you're in inspect mode

Make sure 'background build' is checked.

Hit 'build navmesh'

Testing the NavMesh

Go into test mode

Ctrl-click to create character, or use spawn button

Click to move character around

Congratulations!
