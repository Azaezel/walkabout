Patch notes: R1P3

  • I've added a member of NavMesh called alwaysRender. It will make the navMesh render outside the editor. Could be useful for demos and so on.
  • The tileSize member is now in 'NavMesh options' instead of 'Advanced options'.
  • The performance of rendering voxels in Tile mode is now pretty acceptable!
  • Should work out-of-the-box with Torque 3D MIT 3.0.
