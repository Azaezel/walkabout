Patch notes: R1P2

  • Former versions introduced changes to Engine/source/gfx/primBuilder.cpp. Those changes are no longer required; you can revert that file using your VCS software, or by fetching the file from GitHub.
  • Support for large NavMeshes is officially here! I recommend building them with the nav console open. Some pre-release screenshots have featured a 'time remaining' field in the console; this was removed due to it being completely inaccurate. It will return in future versions!
  • Rendering performance when there is a large NavMesh is pretty poor. I recommend unchecking the 'mesh' tickbox at the top of the nav editor if you have a very large NavMesh in the scene. Just check it on when you need to see the mesh.
  • Rendering performance of a tile's voxels is even worse. Please don't do it.
  • The NavMesh member maxTiles has been removed in favor of calculating this value automatically when a build happens. No more finding out that you didn't set it high enough and some of your mesh didn't build!
